# Öcher Singsang Design Context

## Brand Identity

**Site**: oecher-singsang.de  
**Purpose**: German cultural heritage site preserving Öcher Platt (Aachen dialect)  
**Audience**: Local community, older demographic, heritage enthusiasts  
**Tone**: Warm, traditional, approachable, literary

## Typography System

### Fonts

- **Body**: Vollkorn (German-made serif, "wholegrain")
- **Headings**: Source Sans 3 (clean humanist sans)
- **Rationale**: Serif body + sans headings = traditional/literary feel appropriate for cultural content

### Type Scale (1.414 Augmented Fourth - BOLDER)

| Token         | Size             | Use                 |
|---------------|------------------|---------------------|
| `--text-xs`   | 0.75rem (12px)   | Captions, legal     |
| `--text-sm`   | 0.875rem (14px)  | Metadata, secondary |
| `--text-base` | 1.0625rem (17px) | Body text           |
| `--text-lg`   | 1.25rem (20px)   | Lead paragraphs     |
| `--text-xl`   | 1.75rem (28px)   | h3                  |
| `--text-2xl`  | 2.5rem (40px)    | h2                  |
| `--text-3xl`  | 3.5rem (56px)    | h1 (3.3x body)      |
| `--text-4xl`  | 4.5rem (72px)    | Display/hero        |

### Readability Optimizations

- **Base 17px** (not 16px) - subtle accessibility improvement for older readers
- **Line-height 1.65** - generous vertical rhythm without looking spaced-out
- **Letter-spacing 0.01em** - subtle openness in body text
- **Paragraph margin 1.25em** - clear separation between paragraphs
- **Max-width 65ch** - optimal line length for reading comfort
- **Fluid headings** - clamp() scales with viewport

## Visual Identity

- **Background**: #fdca0c (warm yellow - brand color)
- **Surface**: #fffdf8 (warm off-white / cream paper)
- **Border**: #e8d5b5 (warm tan)
- **Border style**: Double borders on article separators (WordPress heritage)

## Design Constraints

- Content is frozen (rarely/never changes)
- Static 11ty build from WordPress export
- Docker + nginx + Traefik deployment
- Must work well for older visitors without looking geriatric

## Anti-Patterns to Avoid

Per impeccable skill guidelines:

- ❌ Lora, Playfair Display, Cormorant (overused AI defaults)
- ❌ Comic Sans, Papyrus (obviously)
- ❌ Helvetica/Arial for body text
- ❌ Font size below 16px for body
- ❌ Line-height below 1.5 for body
- ❌ Lines longer than 75 characters
